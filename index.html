NewsKyûHEN

PS Vita homebrew contest

I will be participating in the KyûHEN PS Vita homebrew contest. I am currently working on a shoot 'em up game using GameMaker 1.4, so expect a game on the website when I am allowed to share it publicly. The game will be playable on the Vita (homebrew), on Windows and in browsers. I can compile Linux and Mac builds if it is requested. Compiling for Android is also possible albeit quite a bit more complicated. I will also most likely make the code open source on my github.

The game is called Eidetic Sweven, it's a fancy way of saying "vivid dream". So far I have created some graphical assets such as sprites, animations and UI. I have also found royalty free songs that I may use in this project. And I will most likely generate sound effects with bfxr or a similar tool.
